## Consolidación propuesta (clusters por servicio)

- **Diseño web / Ecommerce** ← Diseño Web, Diseño y Desarrollo Web, Diseño Web y Presencia Digital
- **Soluciones digitales** ← Desarrollo a medida, Apps/Software
- **CRM / Automatización** ← Automatización, Email marketing, Estrategia para Pymes (parte)
- **SEO** (PeriSEO) ← Marketing Digital (SEO), Guías SEO, Analítica y Medición
- **Estrategia / Agencia** ← Estrategia Digital, Agencias y Proveedores

Las categorías con 0-1 post se fusionan en su cluster o se eliminan (no generan página).
