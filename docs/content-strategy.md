# Estrategia de contenidos (hub & spoke) — Campero Digital

Basada en el estudio SEO/mercado (junio 2026). Posicionamiento **nacional (España)**, sin foco local. Los servicios son los **hubs**; los artículos del blog son los **spokes** que enlazan al hub correspondiente.

## Clusters

### 1. Diseño web & Ecommerce — hub `/diseno-web/`
Spokes existentes: optimización web móviles, cuánto cuesta una web, diseño UX/UI, página web para psicólogos, web para peluquería, crear con WordPress, mantenimiento web, desarrollo web a medida.
Huecos a cubrir (PeriSEO genera): WooCommerce vs Shopify vs PrestaShop, velocidad y Core Web Vitals para vender, accesibilidad en ecommerce, estructura URL/SEO para tiendas, IA y personalización en ecommerce.

### 2. Soluciones digitales (software/SaaS) — hub `/soluciones-digitales/`
Huecos: software a medida vs SaaS (modelo híbrido), costes de desarrollo a medida 2026, APIs e integraciones, migración de sistemas legados, automatización de procesos internos.

### 3. CRM & Automatización — hub `/crm-automatizacion/`
Spokes existentes: automatización de marketing, email marketing, marketing de contenidos.
Huecos (alta oportunidad, competencia débil): **CRM para ecommerce** (carrito abandonado, segmentación por producto), comparativa HubSpot/Zoho/Clientify, flujos de automatización con ejemplos, RevOps, datos first-party sin cookies.

### 4. SEO (PeriSEO) — hub `/seo/`
Spokes existentes (muchos): auditoría SEO, cómo hacer auditoría paso a paso, agencia SEO pymes, SEO local, posicionamiento abogados, herramientas SEO, SEM/Google Ads.
Nota: el contenido SEO refuerza a PeriSEO; enlazar a periseo.es.

### 5. Accesibilidad web — hub `/accesibilidad-web/`
Huecos (diferenciador puro, casi sin competencia): Ley Europea de Accesibilidad 2026 (a quién obliga, sanciones), WCAG 2.1 vs 2.2, auditoría de accesibilidad paso a paso, accesibilidad en ecommerce, ROI de la accesibilidad.

### 6. Mantenimiento web — hub `/mantenimiento-web/`
Huecos: cada cuánto actualizar WordPress, seguridad web para pymes, backups, qué incluye un plan de mantenimiento.

## Temas prioritarios (gaps de mercado con baja competencia)
1. Accesibilidad WCAG obligatoria 2026 (legal + SEO).
2. CRM para ecommerce / integración CRM-tienda.
3. Software a medida vs SaaS (modelo híbrido).
4. IA en diseño web y personalización.
5. Verifactu para ecommerce (informativo; **no** reclamar conformidad).

## Posicionamiento (mensaje)
"Agencia digital 360: webs que venden + software a medida + CRM y automatización." Diferenciadores: cercanía/partner, integración web+SEO (PeriSEO), accesibilidad, CRM-ecommerce.
